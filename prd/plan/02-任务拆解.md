# Video Downloader 任务拆解

版本：v1.1  
最后更新：2026-06-13  
适用范围：Tauri 桌面版 V1，本机自用，优先 macOS

## 1. Frontend AppShell

负责主窗口与用户交互。

任务：

- 创建主窗口。
- 添加 m3u8 URL 输入框。
- 添加输出目录选择器。
- 添加输出文件名输入框。
- 添加开始、取消、打开文件、文件管理器按钮。
- 添加进度条和状态文本。
- 添加日志面板。
- 根据任务状态控制按钮启用状态。
- 验证输入框键盘删除、复制、粘贴、剪切、替换和全选。

验收：

- 未输入 URL 时开始按钮不可用。
- 运行中输入控件不可编辑。
- 完成后可以打开输出位置。

## 2. Tauri Bridge

负责前后端通信。

任务：

- 实现 `start_download` command。
- 实现 `cancel_download` command。
- 实现 `reveal_path` command。
- 实现 `download-log` event。
- 实现 `download-progress` event。
- 实现 `download-stage` event。

验收：

- 前端可以启动、取消任务。
- 后端日志和阶段变化能实时推送到前端。
- 事件 payload 稳定，便于后续扩展。

## 3. AppState / DownloadSession

负责描述下载任务。

字段：

- id
- input_url
- output_directory
- output_file_name
- output_file_url
- working_directory
- stage
- total_segments
- completed_segments
- log_lines
- error_message
- verification_result
- running_process

验收：

- UI 可根据 stage 更新。
- 日志可持续追加。
- 失败状态可保留错误。
- 同一时间只允许一个 V1 任务运行。

## 4. ToolChecker

负责检查外部工具。

任务：

- 查找 `ffmpeg`。
- 查找 `ffprobe`。
- 查找 `aria2c`。
- 返回工具路径或缺失错误。

验收：

- 工具存在时返回绝对路径。
- 工具缺失时给出安装建议。

## 5. M3U8Resolver

负责把输入 URL 转换为下载清单。

任务：

- 下载远程 m3u8。
- 保存为 `source.m3u8`。
- 校验内容。
- 解析分片 URL。
- 使用 `url::Url::join` 处理绝对 URL、root-relative URL 和 path-relative URL。
- 生成 `full.m3u8`。
- 生成 `aria2.txt`。
- 生成 `concat.txt`。

验收：

- 能解析 200-300 个分片级别的真实 m3u8。
- 分片顺序与源 m3u8 一致。
- 无分片时返回错误。

## 6. ProcessRunner

负责统一执行命令。

任务：

- 用 Rust `std::process::Command` 启动命令。
- 捕获 stdout。
- 捕获 stderr。
- 流式回调日志。
- 返回退出码。
- 支持 terminate。

验收：

- aria2c、ffmpeg、ffprobe 都通过该模块执行。
- 任务取消时外部进程退出。
- 非 0 退出码能返回错误。

## 7. SegmentDownloader

负责分片下载。

任务：

- 调用 aria2c。
- 使用 `aria2.txt`。
- 输出到 `parts/`。
- 解析日志中的完成事件。
- 下载结束后统计 `.ts` 文件数量。
- 检查 `.aria2` 残留。

验收：

- 成功下载所有分片。
- 残留 `.aria2` 时标记失败。
- 日志能显示下载过程。

## 8. MediaMerger

负责合并。

任务：

- 调用 ffmpeg concat。
- 输出 mp4。
- 捕获合并日志。
- 判断退出码。

验收：

- mp4 文件存在且大小大于 0。
- 合并失败时返回 ffmpeg 错误。

## 9. MediaVerifier

负责验证输出。

任务：

- 调用 ffprobe。
- 解析 duration。
- 解析 size。
- 解析 stream codec。
- 解析 width 和 height。

验收：

- 完成态显示媒体信息。
- ffprobe 失败时返回 VerificationFailed。

## 10. ErrorMapper

负责用户可读错误。

错误映射：

- `InvalidURL` -> 请输入有效的 m3u8 URL。
- `MissingTool` -> 缺少必要工具，请安装后重试。
- `InvalidM3U8` -> m3u8 无效或 token 已过期。
- `SegmentDownloadFailed` -> 分片下载未完成。
- `MergeFailed` -> mp4 合并失败。
- `VerificationFailed` -> 输出验证失败。
- `Cancelled` -> 下载已取消。

验收：

- UI 不直接展示 Rust panic 或内部错误堆栈。
- 日志中保留原始命令错误。
