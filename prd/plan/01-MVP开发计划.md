# Video Downloader MVP 开发计划

版本：v1.1  
最后更新：2026-06-13  
适用范围：Tauri 桌面版 V1，本机自用，优先 macOS

## 1. 开发原则

- 先做可用闭环，再做体验优化。
- 复用已验证的命令行流程。
- 当前路线使用 Tauri + Rust core + Web UI。
- 每个阶段都保持 App 可构建、可启动。
- V1 不引入网页解析，避免范围膨胀。

## 2. 里程碑 1：Tauri 项目骨架与主界面

目标：创建可运行的 Tauri 桌面 App 骨架。

任务：

- 创建 Tauri 2 项目结构。
- 实现单窗口 Web UI 页面。
- 添加 URL 输入框、输出目录选择、文件名输入。
- 添加开始、取消、打开文件、在文件管理器中显示按钮。
- 添加状态文本、进度条、日志面板。
- 定义任务状态枚举。
- 建立 Tauri command 和事件通道。

产出：

- `npm run build` 可通过。
- `cargo check` 可通过。
- `npm run tauri dev` 可启动 GUI。
- 用户可以输入 URL 和输出路径。

验收：

- 主窗口不依赖网络即可打开。
- 输入区和日志区布局清晰。
- 输入框支持键盘删除、复制、粘贴、剪切、替换和全选。

## 3. 里程碑 2：工具检测与 m3u8 解析

目标：实现从 m3u8 URL 到分片清单的完整准备流程。

任务：

- 实现工具检测：
  - `ffmpeg`
  - `ffprobe`
  - `aria2c`
- 实现 m3u8 下载。
- 校验 m3u8 内容。
- 解析 `#EXTINF` 后的分片 URL。
- 使用 `url::Url::join` 实现相对 URL 补全。
- 生成：
  - `source.m3u8`
  - `full.m3u8`
  - `aria2.txt`
  - `concat.txt`

产出：

- 输入有效 m3u8 后能生成 4 个任务文件。
- UI 显示分片数量。

验收：

- 绝对分片 URL 正常。
- `/index.php/...` 形式相对分片正常。
- token 无效返回明确错误。

## 4. 里程碑 3：分片下载

目标：用 aria2c 下载所有分片，并把进度反馈给 UI。

任务：

- 实现 Rust ProcessRunner。
- 调用 aria2c。
- 捕获 aria2c stdout/stderr。
- 通过 Tauri event 将日志追加到 UI。
- 解析下载完成数量。
- 下载完成后检查：
  - `.ts` 数量
  - `.aria2` 残留
- 支持取消进程。

产出：

- 有效 m3u8 可下载全部分片。
- UI 能显示下载阶段和日志。

验收：

- 200-300 个分片级别任务可以完整下载。
- 用户取消后进程被终止。
- 下载失败时保留日志和临时目录。

## 5. 里程碑 4：合并与验证

目标：把分片合并成 mp4 并验证输出。

任务：

- 调用 ffmpeg concat 合并。
- 捕获 ffmpeg 日志。
- 合并完成后调用 ffprobe。
- 解析 ffprobe 输出。
- 在 UI 显示：
  - 文件大小
  - 时长
  - 视频编码
  - 分辨率
  - 音频编码

产出：

- 输出可播放 mp4。
- 完成态展示媒体信息。

验收：

- 输出 mp4 可用 QuickTime 或 IINA 播放。
- ffprobe 能返回 duration 和 codec。
- 合并失败时显示错误。

## 6. 里程碑 5：错误处理与体验打磨

目标：让 V1 达到日常自用稳定性。

任务：

- 建立错误分类和用户文案映射。
- 增加重试入口。
- 增加打开输出文件和在文件管理器中显示。
- 增加日志复制能力。
- 增加输出文件已存在提示。
- 增加临时目录路径展示。

产出：

- 常见失败都有明确提示。
- 完成后能快速找到文件。

验收：

- 工具缺失、token 失效、网络失败、合并失败都有可读错误。
- 失败后可以重新开始。
- 文案不暴露过多命令细节，但日志保留原始错误。

## 7. 推荐开发顺序

1. 先实现 Tauri 项目骨架和 Web UI。
2. 建立 Tauri command、事件通道、状态机和日志系统。
3. 用固定 m3u8 URL 完成 Rust 端到端链路。
4. 再把核心流程接到前端状态。
5. 最后处理错误、取消、日志、文件管理器操作。

## 8. V1 完成定义

- 可输入 m3u8 URL。
- 可选择输出目录。
- 可下载所有分片。
- 可合并 mp4。
- 可验证输出文件。
- 可显示错误和日志。
- 可取消任务。
- `npm run build` 通过。
- `cargo check` 通过。
- `npm run tauri dev` 可启动 GUI。
